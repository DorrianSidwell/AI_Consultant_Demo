## ğŸ¨ Screenshot Tips

### 1. Browser Setup
- Use Chrome or Edge in incognito mode (no extensions)
- Set browser to 1920x1080 or similar
- Hide bookmarks bar for cleaner look

### 2. App Setup
- Use a nice theme (not default gray)
- Have realistic sample data loaded
- Use professional-looking document names

### 3. Capturing
- Windows: `Win + Shift + S` for snipping tool
- Mac: `Cmd + Shift + 4` for selection
- Full page: Use browser dev tools for full page capture

### 4. Editing (Optional)
- Add subtle drop shadows
- Crop to focus on the important parts
- Add annotations for complex features
